# MiniPyqtFramework
针对比 helloworld 稍微复杂一点的微小型工具项目

## 前端库
- Pyqt5
- pyuic5.exe

## 后端库
- request
- pathlib
- json

## 多进程/多线程
- subprocess
- ThreadPoolExecutor

## 打包
- pyinstaller

## 开发环境
- Qt Designer
- SublimText
- python3 x32/x64
- win7/win10
